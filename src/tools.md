# Tools

A list of tools I use right now.
Not a polished showcase.
More like the actual workbench: apps I keep open, services that run the studio, podcast gear, CLI tools in the terminal.

## Applications

### Ollama
For testing local models, especially around code, agents, and workflows without depending solely on external APIs.

### Dropbox
Storage and sync for projects, notes, exports, site files, and work folders.

### OpenCode
For testing models, code generation, and automated workflows.

## CLI and dev tools

### git
Version control. Going back. Testing without breaking everything.

### GitHub CLI
Reading issues, creating tickets, inspecting PRs, following checks, and pushing changes faster than going through the UI.

### npm / Node.js
Frontend scripts, tooling, packages, workers, local servers, automations.

### Cloudflare Workers
URL rewrites, cache, edge rules, headers, canonical, routing, and small processing layers in front of sites.

### Tailscale
Private access to machines and services without exposing anything to the public internet.

## Web stack and infra

### Cloudflare
DNS, cache, security rules, redirects, subdomain protection, asset optimization.

### OVH
Legacy shared hosting, DNS/domain management, and some web bricks.

### GitHub
Code, issues, bug tracking, feature ideas, change history.

### Google Analytics 4
Traffic measurement and global site tracking, when quotas and exports do not break everything.

### Search Console
Concrete SEO reading: indexed pages, queries, errors, sitemap, language tags, Google signals.

## Studio Pixel

### Google Ads
Acquisition for Studio Pixel, campaign testing, and local performance tracking.

### Jotform
Client forms, project inquiries, consent, information collection, and file attachments.

### Gmail
Client inquiries, guest artists, reminders, quotes, admin follow-up, quick replies.

### Google Calendar
Studio scheduling, appointments, guests, weekly planning, and internal coordination.

### Instagram
Living portfolio, contests, flash days, stories, announcements, social proof, and direct contact.

### Stripe
Online payments, thinking about deposits, payment links, and potential automation.

### Alma
Pay in 4 installments.

### PostHog
Product analytics for tracking app usage, understanding user paths, and inspecting errors.

## Podcast and content

### Acast
Episode hosting, RSS feed, listening stats.

### Auphonic
Post-production audio, normalization, export, metadata, chapters, and formats.

### DaVinci Resolve
Video editing, export, subtitles, audio mixing, and production.
