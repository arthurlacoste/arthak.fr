# arthak.fr

Site statique minimal.

- source Markdown : `src/`
- sortie HTML : `public/`
- thème : Matrix green
- logo : ASCII `arthak`

## Commandes

```bash
npm install
npm test
npm run build
npm run dev
```

## GitHub Pages

Le workflow `.github/workflows/pages.yml` build `public/` puis déploie sur GitHub Pages.

Dans GitHub : Settings > Pages > Source > GitHub Actions.
